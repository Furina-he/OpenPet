# openpet 用户手册

> 面向普通用户的使用说明。遇到问题先看文末 FAQ；反馈入口见「关于」页。

## 1. 安装

- 系统要求：Windows 10 / 11（64 位）。
- 安装包（NSIS 安装版 / portable 免安装版）目前在打包发布准备中，敬请期待正式发布页。
- 在此之前可从源码运行：安装 [Node.js ≥20](https://nodejs.org) 与 pnpm 9，然后在仓库根目录执行 `pnpm install && pnpm dev`。

## 2. 首次启动引导

第一次启动会出现 4 步引导，约 90 秒：

1. **选语言**——左下角下拉框可切换 简体中文 / English，界面即时切换。
2. **配置模型（连接一个大脑）**——选择一个模型提供商（OpenAI / Claude / Gemini / 通义 / DeepSeek / 自定义中转…），粘贴 API Key，点「拉取模型列表」后挑一个模型；没有 Key 也可以选本地 Ollama，或先「暂时跳过」。
3. **选角色**——默认伙伴即可开始，之后可在角色库导入更多角色。
4. **说第一句话**——点一句预设话术，角色就会回应你。

引导可随时点「跳过」，之后在 Hub 设置里都能补配。

## 3. 日常使用

- **托盘**：右下角托盘图标——左键 显示/隐藏角色，双击 打开聊天，中键 切换鼠标穿透，右键 完整菜单（不打扰 / 打开 Hub / 退出）。
- **热键**（可在 Hub → 系统 → 热键 修改）：呼出聊天、显示/隐藏角色、鼠标穿透、不打扰、打开 Hub。
- **鼠标穿透**：开启后点击会穿过角色落到桌面，角色不再挡操作；再次触发热键或托盘中键恢复互动。
- **聊天气泡**：角色上方的浮层气泡显示回复；长文自动折叠，点「展开」看全文。
- **语音**：聊天输入框的麦克风按钮=点击开始录音、再点结束并转写；在 系统 → 通用 里开启「自动朗读回复」可让角色读出回复（需先配置 TTS 模型）。
- **右键角色**：快捷菜单（聊天 / 穿透 / 显隐 / 设置）。

## 4. 角色包

- 在 Hub → 角色 → 角色库 点「导入 .dspack」或「导入文件夹」，确认摘要后即可加入；**双击卡片**热切换当前角色。
- 支持 VRM（3D）与 Live2D（Cubism 4/5）两类角色。
- **Live2D 角色需要自行下载 Cubism Core**（`live2dcubismcore.min.js`，Live2D 官网免费提供），按提示放入指定目录；缺失时角色会降级为占位表情，不影响其它功能。

## 5. 数据与隐私

- 所有数据（对话、记忆、角色、知识库、配置）保存在本地 `%APPDATA%/openpet`（Hub → 系统 → 数据 → 「打开数据目录」可直达）。
- **备份**：数据页可导出 `.dsbak` 备份（含对话历史、人设演进、长期记忆；不含 API Key）；导入 `.dsbak` 会在下次启动时整体替换当前数据，旧数据自动备份为 `.bak`。
- **API Key 以明文随配置保存在本地**，不会上传到任何服务器；请勿把数据目录或备份分享给他人。
- 长期记忆、知识库 RAG、剪贴板/截屏等系统访问均有独立开关（Hub → 系统 → 隐私），默认最小化。

## 6. FAQ

**Q1：我的 API Key 存在哪里？安全吗？**
存在本地数据目录的配置文件里，明文保存、不上传。请像保管密码一样保管你的数据目录与 `.dsbak` 备份；如怀疑泄露，去提供商后台重置 Key。

**Q2：导入 Live2D 角色后是一张占位脸？**
缺少 Cubism Core 运行库。到 Live2D 官网下载 `live2dcubismcore.min.js`，按角色库提示放入对应目录后重启即可。

**Q3：鼠标穿透开了以后点不到角色了，怎么恢复？**
这是穿透的正常效果。用托盘图标中键、托盘右键菜单或全局热键（默认见 热键 页）再切一次即可恢复互动。

**Q4：怎么把数据搬到新电脑？**
旧电脑：Hub → 系统 → 数据 → 导出 `.dsbak`。新电脑：安装后在同一页面导入该文件并重启。注意 API Key 不在备份里，需要重新填写。

**Q5：杀毒软件报毒 / 拦截怎么办？**
未签名的 Electron 应用可能被误报。请从官方发布页下载安装包，校验无误后在杀软中添加信任；正式版会逐步加入代码签名以减少误报。

**Q6：回复报错「钥匙好像不对 / 额度用完」是什么意思？**
分别对应 API Key 无效（去 模型 API 页重填并保存）和提供商限流/欠费（换个模型或稍后再试）。详细错误可在开启开发者模式后于 诊断 页查看。
