## 2. 设计系统基础

### 2.1 色彩

| 角色 | 浅色（默认） | 深色（备选） | 用途 |
| --- | --- | --- | --- |
| 玻璃面板底 | `rgba(255, 255, 255, 0.72)` | `rgba(18, 20, 28, 0.62)` | 所有浮层底色 |
| 玻璃描边 | `rgba(0,0,0,0.06)` 1px | `rgba(255,255,255,0.10)` 1px | 内外边界 |
| 主文 | `#171821` | `#F2F3F8` | 标题/正文 |
| 次文 | `rgba(23,24,33,0.55)` | `rgba(242,243,248,0.62)` | 辅助说明 |
| **角色暖色（品牌锚点）** | `#FFB4A2 → #FF8FAB` 线性渐变 | 同 | 气泡、情绪点亮、CTA |
| 情绪辅色 | `#FFD166`（喜）`#7FB7FF`（思）`#C6A8FF`（羞） | 同 | 表情驱动的微光 |
| 科技冷点 | `#6FA8FF` | 同 | 连接/Token/状态指示 |
| 语义 · 成功 | `#7FE3A1` | 同 | |
| 语义 · 警告 | `#FFB454` | 同 | |
| 语义 · 错误 | `#FF6B7A` | 同 | |
| 语义 · 降级 | `#C8924A` 暗琥珀 | 同 | 离线兜底/演示模式横条 |

> 主色调克制：80% 中性玻璃 + 15% 暖色（角色相关）+ 5% 冷色（系统状态）。

### 2.2 玻璃效果

**浅色主题（默认）**

```
backdrop-filter: blur(28px) saturate(180%)
background: rgba(255,255,255,0.72)
border: 1px solid rgba(0,0,0,0.06)
box-shadow:
  0 8px 32px rgba(0,0,0,0.08),
  inset 0 1px 0 rgba(255,255,255,0.55)
```

**深色主题（备选）**

```
backdrop-filter: blur(28px) saturate(180%)
background: rgba(18,20,28,0.62)
border: 1px solid rgba(255,255,255,0.10)
box-shadow:
  0 16px 48px rgba(0,0,0,0.36),
  inset 0 1px 0 rgba(255,255,255,0.08)
```

两种主题都在角色周围浮层额外叠一层径向暖色高光（10% 透明度），实现"角色辐射温度"。

> 主题切换：在 D4 显示与窗口 → "界面主题" 中切换 `跟随系统 / 浅色 / 深色`，默认 `跟随系统`，系统未指明时降级为浅色。

### 2.3 字体

- 中文：**苹方 / PingFang SC** 或 **思源黑体**（Light / Regular / Medium）
- 英文：**Inter**（300/400/500/600）
- 数字与代码：**JetBrains Mono**（Token 计数、API Key、日志）
- 字号阶梯：`12 / 13 / 14 / 16 / 20 / 28 / 36`
- 行高：正文 1.5 / 标题 1.25

### 2.4 圆角与间距

- 圆角：按钮 `8` / 输入框 `10` / 卡片 `12` / 浮层 `16` / 气泡 `18`（左下/右下不对称呈"说话"感）
- 间距栅格：`4 / 8 / 12 / 16 / 24 / 32 / 48`

### 2.5 图标 & 动效

- 图标：**Lucide** 风格，1.5px 描边、圆角端点；角色相关图标允许手绘暖色点缀
- 通用过渡：`200ms cubic-bezier(0.22, 1, 0.36, 1)`
- 角色表情切换：`350–500ms` 平滑 BlendShape 过渡
- 流式文本：逐字 `30–50ms`，伴随表情实时驱动
- 微互动：按钮按压 `scale(0.97)` + 暖色辉光闪一下

### 2.6 通用组件库

#### 2.6.1 基础组件

- `GlassPanel` 三档：S 浮窗、M 浮层、L 设置窗
- `Button` 主/次/幽灵/危险 四态 × 三尺寸
- `Input / Select / Slider / Switch / Tag / Badge`
- `Avatar`（角色头像，圆形 + 情绪环）
- `Bubble`（左/右两向，含流式光标）
- `EmotionDot`（情绪标签彩点）
- `KeyCap`（热键展示，模拟物理键帽阴影）

#### 2.6.2 抽屉规范（统一三档）

| 类型 | 宽度 | 用途 | 例 |
| --- | --- | --- | --- |
| 列表型 | `280px` | 侧边导航/会话列表 | B3 会话历史、Hub 左导 |
| 表单型 | `420px` | 配置/编辑 | C 引导、确认弹窗 |
| 详情型 | `560px` | 详情/编辑器 | E2 角色详情、G2 插件详情 |

#### 2.6.3 Toast 规范

| 形态 | 位置 | 尺寸 | 持续时间 | 用途 |
| --- | --- | --- | --- | --- |
| 顶栏薄条 | Hub Window 顶部 | 全宽 × 32px | 600ms | 设置即时保存 `✓ 已保存` |
| 浮卡 | 桌面右下 | `280 × 56` | `3s` 自动消失 | 模式切换、网络恢复 |
| 长 toast | 桌面右下 | `360 × 80` | `5s` + undo 按钮 | 删除可撤销 |

堆叠规则：浮卡同时最多 3 条，新进入挤掉最早的；点击外部关闭。

### 2.7 角色 HUD 通道

> 角色本身就是状态指示器。下表定义"角色身上的 5 个 UI 槽位"，各模块向这些槽位写入信号，避免互相覆盖。

| 槽位 | 尺寸 | 用途 | 优先级 | 例 |
| --- | --- | --- | --- | --- |
| 头顶 | `12×12` 徽标 | 网络/连接相关 | P3 | J4 离线 ✈、I2 桥接消息 |
| 肩膀（右锁骨） | `14×14` 徽标 | DND / 模式 | P2 | A4 月牙 |
| 胸口 | `8×8` 暖色脉冲点 | 情绪持久态 | P4 | F4 关系等级亮起 |
| 脚下 | 4px 高色条 / 红点 | 系统/穿透/工具调用 | P1 | A3 穿透色条、G2 高风险红点 |
| 全身姿态 | 整体 BlendShape | AI 情绪/失败 | P0 | J3 歪头、欢喜跳 |

冲突规则：高优先级槽位长时间占用时压低同时段低优先级表达；最多同时显示 2 个非全身槽位。

### 2.8 破坏性操作统一规范

| 等级 | 触发示例 | 确认方式 |
| --- | --- | --- |
| ① 可恢复 | F3 删一条记忆、B3 删会话 | undo toast 5s（直接执行 + 撤销窗口） |
| ② 不可恢复但低代价 | E1 卸载角色、G1 卸载插件 | 二次确认弹窗，列出影响 |
| ③ 不可恢复且高代价 | D7 清空对话、清空记忆、卸载所有数据 | 二次确认 + 输入 `DELETE` 字样 |

红视觉规则：`等级②/③` 的入口按钮使用错误色描边；二次确认弹窗整张红色描边。

## 3. 信息架构与导航

### 3.1 三层结构

```
┌─────────────────────────────────────────────┐
│ Layer 3 · 窗口层（Hub Window）              │  设置/角色/插件/知识库
│   一个统一窗口，左侧导航 + 内容区            │
├─────────────────────────────────────────────┤
│ Layer 2 · 浮层（Floating）                   │  聊天浮层、引导、确认弹窗
│   贴附桌面或角色，半模态                     │
├─────────────────────────────────────────────┤
│ Layer 1 · 桌面层（Desktop）                  │  角色本体 + 桌面气泡 + 微指示
│   常驻、透明、可穿透                          │
└─────────────────────────────────────────────┘
                   ↑ 系统托盘 / 全局热键
```

用户 90% 时间只看到 Layer 1；Layer 2 按需召唤；Layer 3 是低频管理界面。**Layer 3 不抢 Layer 1 的视觉戏份。**

### 3.2 入口关系

| 目标 | 路径 |
| --- | --- |
| 唤起聊天 | 点击角色 / `Ctrl+Shift+D` / 托盘菜单 |
| 打开 Hub | 托盘 → 设置 / 聊天浮层右上 ⚙ / 角色右键 / `Ctrl+Shift+,` |
| 切换角色（快） | 角色右键 → 切换 |
| 切换角色（慢） | Hub → 角色 → 角色库 |
| 退出/隐藏 | 托盘菜单 |

### 3.3 Hub Window 导航

```
┌──────────────┬─────────────────────────────────────┐
│  ⌂ 总览       │ 顶栏 56px：当前角色 · 状态 · 全局搜索│
│  ◉ 角色  ▾   │                                     │
│   ├ 角色库    │                                     │
│   ├ 编辑器    │       ▶ 当前选中页内容              │
│   └ 商店      │                                     │
│  ✎ 对话  ▾   │                                     │
│   ├ 历史      │                                     │
│   ├ 记忆      │                                     │
│   └ 人格      │                                     │
│  ⚡ 模型 API  │                                     │
│  ⊞ 插件       │                                     │
│  ☷ 知识库     │                                     │
│  ⇄ IM 桥接    │                                     │
│  ⚙ 系统  ▾   │                                     │
│   ├ 显示窗口  │                                     │
│   ├ 语音      │                                     │
│   ├ 热键      │                                     │
│   ├ 隐私      │                                     │
│   ├ 数据      │                                     │
│   └ 关于      │                                     │
├──────────────┴─────────────────────────────────────┤
│ 状态条：● Claude 已连接 · 内存 320MB · 在线         │
└─────────────────────────────────────────────────────┘
窗口默认 1080 × 720，min 880 × 600
```

- 左侧导航 280px，可折叠为 64px 仅图标
- 顶栏 56px：左侧当前角色头像（hover 下拉切换），中部面包屑/搜索，右侧最小化/关闭
- 状态条 32px：连接/内存/网络/当前模型；不显示 Token（避免与 B5/F2 冗余）

### 3.4 桌面层与浮层附着规则

聊天浮层默认贴在角色斜上方 16px，跟随角色拖动；靠近屏幕边缘时自动翻边。**桌面气泡（A2）和聊天浮层（B1）不同时显示**——浮层打开时桌面气泡自动收回。

```

                  ╭──────────────────────╮
                  │       ┌───┐         │
```
